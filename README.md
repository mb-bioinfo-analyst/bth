# Bioinformatics Tools Hub

A browser-based collection of bioinformatics utilities for DNA, RNA, and protein sequence analysis.

## Features

- Sequence conversion
- FASTA tools
- ORF finding
- Motif search
- k-mer analysis
- Codon usage tools
- Restriction enzyme analysis
- Protein property analysis

## Privacy

All sequence processing runs locally in the browser.

## Tech Stack

- React
- TypeScript
- Vite
- TailwindCSS
